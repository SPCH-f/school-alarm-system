# 🏫 School Alarm System (ระบบแจ้งเตือนตารางเรียนอัตโนมัติ)

![C#](https://img.shields.io/badge/c%23-%23239120.svg?style=for-the-badge&logo=c-sharp&logoColor=white)
![.Net](https://img.shields.io/badge/.NET-5C2D91?style=for-the-badge&logo=.net&logoColor=white)
![Windows](https://img.shields.io/badge/Windows-0078D6?style=for-the-badge&logo=windows&logoColor=white)

โปรแกรมสำหรับตั้งเวลาและเล่นเสียงกริ่งโรงเรียนอัตโนมัติ พัฒนาด้วย **C# (Windows Forms)** ออกแบบมาเพื่อให้คุณครูและฝ่ายโสตทัศนศึกษาใช้งานได้ง่าย เพียงแค่ตั้งเวลาและเลือกไฟล์เสียง โปรแกรมจะจัดการแจ้งเตือนให้ตรงเวลาเป๊ะ พร้อมฟีเจอร์ทำงานเบื้องหลังโดยไม่รบกวนการใช้งานคอมพิวเตอร์

---

## Tech Stack

- **Language:** C#
- **Framework:** .NET / Windows Forms (WinForms)
- **Data Storage:** JSON 
- **System Integration:** Windows Registry API

---

## Installation

โปรแกรมนี้เป็นแบบ Portable ไม่จำเป็นต้องติดตั้งลงเครื่อง:
1. ไปที่เมนู **[Releases](../../releases)** ของ Repository นี้
2. ดาวน์โหลดไฟล์ `SchoolAlarmUI.exe`
3. นำไปวางในโฟลเดอร์ที่ต้องการ (เช่น `D:\SchoolAlarm\`) แล้วดับเบิ้ลคลิกเพื่อเปิดใช้งานได้ทันที!

---

## How to Use

### 1. การจัดการตารางเวลา
- **เพิ่มกิจกรรม:** ไปที่เมนู `Alarm` > `เพิ่มกิจกรรม` (หรือคลิกที่แถวว่างบรรทัดล่างสุด) แล้วพิมพ์เวลาและชื่อคาบเรียน
- **เลือกไฟล์เสียง:** ดับเบิ้ลคลิกที่ช่อง "ไฟล์เสียง" เพื่อเปิดหน้าต่างเบราว์เซอร์สำหรับเลือกไฟล์เสียงกริ่ง
- **ลบกิจกรรม:** คลิกเลือกแถวที่ไม่ต้องการ แล้วไปที่ `Alarm` > `ลบกิจกรรม`
- **ทดสอบเสียง:** คลิกเลือกคาบเรียน แล้วไปที่ `Alarm` > `ทดสอบเสียง`

### 2. การบันทึกข้อมูล
- ข้อมูลจะบันทึกอัตโนมัติ หรือสามารถกดบันทึกเองได้ที่ `File` > `บันทึกข้อมูล` (ข้อมูลจะถูกเซฟเป็นไฟล์ `schedule.json` ไว้ข้างๆ ตัวโปรแกรม)

### 3. การตั้งค่าขั้นสูง (Options)
- **เปิดทำงานอัตโนมัติ:** ไปที่เมนู `Options` > คลิก `เปิดโปรแกรมอัตโนมัติเมื่อเปิดเครื่อง` (จะมีเครื่องหมาย ✓ ปรากฏขึ้น)
- **ซ่อนโปรแกรม:** หากกดปุ่มย่อหน้าต่าง (-) ตัวโปรแกรมจะหายไปจาก Taskbar และไปซ่อนตัวอยู่ที่ System Tray (ใกล้ๆ นาฬิกาของ Windows) ให้ดับเบิ้ลคลิกที่ไอคอนเพื่อเรียกหน้าต่างกลับมา
